# Community

## Getting Help

- **GitHub Issues**: Report bugs and request features
- **GitHub Discussions**: Ask questions and share ideas
- **Contributing**: See [CONTRIBUTING.md](../CONTRIBUTING.md)

## Reporting Bugs

Please include:
1. Blaze version (`cargo pkgid`)
2. Rust version (`rustc --version`)
3. Minimal reproduction SQL
4. Expected vs actual behavior

## Feature Requests

Open a GitHub issue with the `enhancement` label. Include:
1. Use case description
2. Proposed API (if applicable)
3. Alternatives considered

## Roadmap

See the [GitHub Projects board](https://github.com/blaze-db/blaze/projects) for planned features.
